09

Takeaways

What worked

  • Lambda-deferred state reads collapsed recomp count to one per visible button.
  • drawWithCache eliminated per-frame Brush allocation in scroll.
  • Deferring SKSL background compile saves ~60–80 ms of TTID on cold start.
  • Baseline profile + R8 release variant: 38% faster cold start vs Phase 2.

Trade-offs

  • Per-frame draw cost is now the dominant scroll bottleneck.
  • Glow shadow + haze blur + inner shadow + shimmer border + radial overlay all run every frame.
  • Idle P95/P99 tail latency drifted slightly upward post-recomp (P99 +24%).

What we learned

  • Thermal state dominates short A/B comparisons — always cool the device.
  • BaseInteractiveButton SumMs is a reliable thermal canary.
  • The unrelated registry lock was protecting a race that can't happen — dropped safely.
